GoogleDrive

Use to login google drive and get data from google drive api. You need to add https://apis.google.com/js/api.js to html.

Props

  • client_id(required): You can see here to find your client_id.
  • updateSigninStatus(required): This will be called when status of auth is checked.
  • children(required)
  • authClick: If isSignedIn is false, onClick of children will be this function.
  • signoutClick: If isSignedIn is true, onClick of children will be this function.
